## 🏗️ 技术架构

### 前端技术
- React 18 + TypeScript
- Tailwind CSS（样式）
- React Router（路由）
- Recharts（图表）
- Axios（HTTP 客户端）
- Monaco Editor（代码编辑器）

### 后端技术
- FastAPI（Web 框架）
- SQLAlchemy（ORM）
- PostgreSQL（数据库）
- Redis（缓存）
- 阿里云 OSS（对象存储）

### 新增数据库表
```sql
-- 课程表
courses
- id, title, subtitle, description
- cover_image_url, price, original_price
- difficulty_level, status
- instructor_id, created_at, updated_at

-- 章节表
chapters
- id, course_id, title, description
- sort_order, created_at

-- 课时表
lessons
- id, chapter_id, title, lesson_type
- video_url, video_duration, content
- sort_order, is_free_trial, created_at

-- 练习题目表
exercises
- id, lesson_id, title, description
- starter_code, solution_code
- test_cases, difficulty, points
- created_at

-- 学习进度表
enrollments
- id, user_id, course_id, progress
- completed_lessons, last_lesson_id
- started_at, completed_at

-- 订单表
orders
- id, user_id, course_id, amount
- status, payment_method, payment_time
- created_at
```
