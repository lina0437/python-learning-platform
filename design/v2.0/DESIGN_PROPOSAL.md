# Python 学习平台 v2.0 设计方案

**版本:** 2.0  
**设计日期:** 2026-03-24  
**设计风格:** 现代、专业、友好、高效  
**目标用户:** Python 初学者、转行者、技能提升者

---

## 🎨 一、整体设计风格

### 设计理念
> **"让学习变得简单而有趣"**

- **专业感** - 建立信任，体现教学质量
- **友好性** - 降低学习焦虑，鼓励探索
- **清晰度** - 信息层次分明，减少认知负担
- **激励性** - 可视化进度，增强成就感

### 设计关键词
```
现代 Modern | 简洁 Clean | 活力 Energetic | 可信 Trustworthy
```

## 🎨 二、配色方案

### 主色调

| 颜色 | 色值 | 用途 |
|------|------|------|
| 🟦 主蓝 | `#3B82F6` | 主按钮、链接、重要操作 |
| 🟩 成功绿 | `#10B981` | 完成状态、正确提示、价格 |
| 🟨 警告黄 | `#F59E0B` | 提醒、注意、中等难度 |
| 🟥 错误红 | `#EF4444` | 错误提示、高危操作 |
| 🟪 品牌紫 | `#8B5CF6` | 高级功能、VIP 标识 |

### 中性色

| 颜色 | 色值 | 用途 |
|------|------|------|
| 深灰 | `#1F2937` | 主标题、重要文字 |
| 中灰 | `#6B7280` | 副标题、说明文字 |
| 浅灰 | `#9CA3AF` | 占位符、禁用状态 |
| 背景 | `#F9FAFB` | 页面背景 |
| 白色 | `#FFFFFF` | 卡片背景、内容区 |

### 渐变色

```css
/* Hero 背景渐变 */
hero-gradient: linear-gradient(135deg, #667EEA 0%, #764BA2 100%)

/* 按钮渐变 */
button-gradient: linear-gradient(135deg, #3B82F6 0%, #8B5CF6 100%)

/* 卡片悬停渐变 */
card-hover: linear-gradient(135deg, #F0F9FF 0%, #E0F2FE 100%)
```
