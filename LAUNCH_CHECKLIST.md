# 项目启动清单

## 📋 启动前检查

- [ ] 安装 Docker & Docker Compose

### 2. 代码仓库

- [ ] 创建 GitHub 仓库

### 3. 基础设施

- [ ] 注册 Supabase 账号，创建项目

## 🚀 部署流程

### Staging 环境

1. 创建 staging 分支
2. 自动部署到 staging 环境
3. 内部测试验证
4. 性能测试

### Production 环境

1. 从 main 分支创建 release tag
2. 自动部署到 production
3. 灰度发布 (10% → 50% → 100%)
4. 监控关键指标
5. 如有问题，快速回滚
