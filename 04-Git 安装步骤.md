# 🌿 Git 安装步骤（零基础版）

> 预计耗时：5 分钟

---

## 步骤 1：访问 Git 官网

1. 打开浏览器
2. 在地址栏输入：
   ```
   https://git-scm.com/
   ```
3. 按回车

---

## 步骤 2：下载 Git

1. 页面加载后，你会看到一个**红色按钮**
2. 按钮上写着 "Download for Windows"
3. **点击这个按钮**
4. 浏览器会开始下载（文件约 50MB）
5. 等待下载完成

---

## 步骤 3：运行安装程序

1. 找到下载的文件
   - 通常在 "下载" 文件夹
   - 文件名类似：`Git-2.x.x-64-bit.exe`

2. **双击**这个文件

---

## 步骤 4：安装设置（重要）

### 4.1 欢迎界面
- 点击 "Next"

### 4.2 安装位置
- 保持默认（C:\Program Files\Git\）
- 点击 "Next"

### 4.3 选择组件
- **全部保持默认**（所有勾选都不要改）
- 点击 "Next"

### 4.4 开始菜单快捷方式
- 保持默认
- 点击 "Next"

### 4.5 命令行编辑器（⚠️ 注意）
- 会看到一个下拉菜单，默认是 "Vim"
- **保持默认 "Vim" 就可以**
- 点击 "Next"

### 4.6 初始分支名
- 保持默认（main 或 master）
- 点击 "Next"

### 4.7 调整 PATH 环境（⚠️ 重要）
- 选择 **第二个选项**：
  ```
  Git from the command line and also from 3rd-party software
  ```
- 点击 "Next"

### 4.8 SSH 程序
- 保持默认 "Use bundled OpenSSH"
- 点击 "Next"

### 4.9 HTTPS 传输后端
- 保持默认 "Use the OpenSSL library"
- 点击 "Next"

### 4.10 行尾转换（⚠️ 注意）
- 选择 **第一个选项**：
  ```
  Checkout Windows-style, commit Unix-style line endings
  ```
- 点击 "Next"

### 4.11 终端模拟器
- 保持默认 "Use MinTTY"
- 点击 "Next"

### 4.12 额外功能
- 保持默认（全部勾选）
- 点击 "Next"

### 4.13 实验性功能
- 全部不勾选
- 点击 "Next"

### 4.14 开始安装
- 点击 "Install"
- 如果弹出用户账户控制，点击 "是"

---

## 步骤 5：等待安装完成

1. 安装进度条会开始走动
2. 大约需要 1-3 分钟
3. 等待直到看到 "Completing..."
4. 点击 "Finish"

---

## 步骤 6：验证安装

### 6.1 打开命令提示符

1. 按 **Win + R** 键
2. 输入：`cmd`
3. 按回车

### 6.2 检查 Git 版本

输入：
```
git --version
```

按回车，应该显示：
```
git version 2.x.x.windows.1
```

---

## 步骤 7：配置 Git（首次使用）

继续在命令提示符中输入以下命令（逐行输入，每行按回车）：

### 7.1 设置你的名字

```
git config --global user.name "你的名字"
```

> 把 "你的名字" 换成你的真实姓名或昵称

### 7.2 设置你的邮箱

```
git config --global user.email "你的邮箱@example.com"
```

> 把邮箱换成你的真实邮箱（没有 GitHub 邮箱的话，先用 QQ 邮箱或其他）

### 7.3 验证配置

输入：
```
git config --list
```

应该能看到你刚才设置的名字和邮箱

---

## ✅ 安装成功检查

如果 `git --version` 显示了版本号，说明安装成功！🎉

---

## 🐛 常见问题

### 问题 1：命令不识别

如果显示 `'git' 不是内部或外部命令`

**解决**：
1. 关闭命令提示符窗口
2. 重新打开一个新的
3. 再试一次

### 问题 2：安装时卡住

**解决**：
- 耐心等待，Git 安装有时比较慢
- 如果超过 10 分钟，重启电脑重新安装
