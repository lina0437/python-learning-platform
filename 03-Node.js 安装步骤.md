# 📦 Node.js 安装步骤（零基础版）

> 预计耗时：10 分钟

---

## 步骤 1：打开浏览器

1. 找到你电脑上的浏览器（Chrome、Edge 都可以）
2. 点击打开

---

## 步骤 2：访问 Node.js 官网

1. 在浏览器地址栏（最上面的长条框）点击一下
2. 删除里面的内容
3. 输入以下网址：
   ```
   https://nodejs.org/
   ```
4. 按 **Enter（回车）键**

---

## 步骤 3：下载 Node.js

1. 页面加载后，你会看到**两个绿色按钮**：
   - 左边：**LTS**（推荐，长期支持版）← 选这个！
   - 右边：Current（当前版）

2. **点击左边的 LTS 按钮**（例如：Download Node.js 20.x.x LTS）

3. 浏览器会开始下载（底部或右上角会显示下载进度）

4. 等待下载完成

---

## 步骤 4：运行安装程序

1. 下载完成后，找到下载的文件
   - 通常在 "下载" 文件夹
   - 文件名类似：`node-v20.x.x-x64.msi`

2. **双击**这个文件

---

## 步骤 5：安装设置

### 5.1 欢迎界面
- 点击 "Next"

### 5.2 许可协议
- 勾选 "I accept the terms..."
- 点击 "Next"

### 5.3 安装位置
- 保持默认（C:\Program Files\nodejs\）
- 点击 "Next"

### 5.4 自定义安装
- 保持默认（所有功能都安装）
- 点击 "Next"

### 5.5 安装工具
- 如果提示是否安装 "Tools for Native Modules"
- 选择 **"No"**（我们暂时不需要）
- 点击 "Next"

### 5.6 开始安装
- 点击 "Install"
- 如果弹出用户账户控制，点击 "是"

---

## 步骤 6：等待安装完成

1. 安装进度条会开始走动
2. 大约需要 2-5 分钟
3. 等待直到看到 "Finished"
4. 点击 "Finish"

---

## 步骤 7：验证安装

### 7.1 打开命令提示符

1. 按键盘上的 **Win + R** 键
2. 在弹出的框里输入：`cmd`
3. 按 **Enter（回车）键**
4. 会出现一个黑色窗口

### 7.2 检查 Node.js

在黑色窗口中输入：
```
node --version
```
按回车，应该显示：`v20.x.x` 或类似版本

### 7.3 检查 npm

继续输入：
```
npm --version
```
按回车，应该显示：`10.x.x` 或类似版本

---

## ✅ 安装成功检查

如果两个命令都显示了版本号：

```
node --version
v20.11.0

npm --version
10.2.4
```

**恭喜你！Node.js 安装成功！** 🎉

---

## 🐛 常见问题

### 问题 1：命令不识别

如果显示 `'node' 不是内部或外部命令`

**解决**：
1. 关闭命令提示符窗口
2. 重新打开一个新的
3. 再试一次

### 问题 2：安装失败

**解决**：
1. 重启电脑
2. 重新运行安装程序
3. 如果还不行，告诉我错误信息
