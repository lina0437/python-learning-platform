## 🛠️ 第一步：检查你的电脑

### 1. 查看 Windows 版本

```
步骤：
1. 按键盘上的 Win+R 键（Win 键是左下角那个微软标志的键）
2. 在弹出的框里输入：winver
3. 按回车
4. 会显示 Windows 版本，告诉我是什么
```

### 2. 查看电脑配置

```
步骤：
1. 找到桌面上的"此电脑"或"我的电脑"图标
2. 右键点击它
3. 选择"属性"
4. 找到"内存"或"RAM"，告诉我数字是多少
```

### 3. 检查磁盘空间

我们需要至少 **20GB** 的可用空间

```
步骤：
1. 打开"此电脑"
2. 看 C 盘（或你要安装的盘）
3. 右键点击 → 属性
4. 看"可用空间"有多少
```
