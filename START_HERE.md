## 🎯 项目愿景

**不是做一个产品，而是打造一条"产品生产线"**

第一个 Python 交互式学习网站是验证"产品矩阵加速器"引擎的原型产品。真正的价值在于后续能指数级加速第二、第三、第 N 个产品的开发。

### 核心原则

1. ✅ **用户体验优先** - 预算充足，不省钱牺牲体验
2. ✅ **性能目标** - 支持 1000 人同时在线
3. ✅ **产品线思维** - 通用引擎 + 插件架构
4. ✅ **商业化导向** - 从第一天就按赚钱产品标准打造

## 🛠️ 技术栈

### 前端
- **框架**: React 18 + TypeScript
- **构建**: Vite 5
- **样式**: TailwindCSS 3
- **编辑器**: Monaco Editor (VS Code 同款)
- **状态**: React Query + Context

### 后端
- **框架**: FastAPI (Python 3.11)
- **数据库**: PostgreSQL (Supabase)
- **缓存**: Redis
- **认证**: JWT
- **ORM**: SQLAlchemy 2

### 基础设施
- **部署**: Railway / AWS
- **容器**: Docker + Compose
- **代码执行**: Judge0 (自托管)
- **监控**: Sentry + Better Stack
- **CDN**: Cloudflare
