# 🛠️ 环境搭建指南

> 按照以下步骤完成开发环境配置，预计耗时 60 分钟

---

## 步骤 1：安装 Python 3.10+

### Windows 用户

1. 访问 Python 官网：https://www.python.org/downloads/
2. 下载 **Python 3.10** 或更高版本
3. 运行安装程序
4. ⚠️ **重要**：勾选 "Add Python to PATH"
5. 点击 "Install Now"

### 验证安装

打开命令提示符（Win+R，输入 `cmd`），运行：

```bash
python --version
```

应该显示：`Python 3.10.x` 或更高

---

## 步骤 2：安装 Node.js 18+

### Windows 用户

1. 访问 Node.js 官网：https://nodejs.org/
2. 下载 **LTS 版本**（推荐 18.x 或 20.x）
3. 运行安装程序
4. 一直点击 "Next" 即可

### 验证安装

```bash
node --version
npm --version
```

应该显示：`v18.x.x` 和 `9.x.x`

---

## 步骤 3：安装 Docker Desktop

### Windows 用户

1. 访问 Docker 官网：https://www.docker.com/products/docker-desktop/
2. 下载 Docker Desktop for Windows
3. 运行安装程序
4. 安装完成后**重启电脑**
5. 启动 Docker Desktop
6. 等待右下角图标变为绿色（运行中）

### 验证安装

```bash
docker --version
docker-compose --version
```

---

## 步骤 4：安装 VS Code（代码编辑器）

1. 访问 VS Code 官网：https://code.visualstudio.com/
2. 下载并安装
3. 安装以下扩展（在 VS Code 左侧扩展面板搜索安装）：
   - Python (Microsoft)
   - Pylance
   - ES7+ React/Redux/React-Native snippets
   - Tailwind CSS IntelliSense
   - Docker
   - GitLens

---

## 步骤 5：安装 Git（版本控制）

1. 访问 Git 官网：https://git-scm.com/
2. 下载并安装
3. 使用默认设置即可

### 验证安装

```bash
git --version
```

### 配置 Git（首次使用）

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

---

## 步骤 6：创建 GitHub 账号（代码托管）

1. 访问 https://github.com/
2. 点击 "Sign up" 注册账号
3. 验证邮箱
4. 记住你的用户名和密码

---

## 🐛 常见问题

### 问题 1：Python 命令不识别

**原因**：安装时没有勾选 "Add Python to PATH"

**解决**：
1. 重新运行 Python 安装程序
2. 选择 "Modify"
3. 勾选 "Add Python to PATH"
4. 完成安装

### 问题 2：Docker Desktop 无法启动

**原因**：WSL2 未启用或 BIOS 虚拟化未开启

**解决**：
1. 以管理员身份打开 PowerShell，运行：
   ```powershell
   wsl --install
   ```
2. 重启电脑
3. 如果还不行，进入 BIOS 开启虚拟化（Intel VT-x 或 AMD-V）

### 问题 3：下载速度慢

**解决**：使用国内镜像
- Python: https://mirrors.aliyun.com/python/
- Node.js: https://npm.taobao.org/mirrors/node/
- Docker: 配置 Docker Hub 镜像加速器
