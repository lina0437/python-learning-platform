# 🐍 Python 安装详细步骤（零基础版）

> 按照这个步骤，100% 能安装成功！
> 
> **预计耗时：15 分钟**

---

## 步骤 1：打开浏览器

1. 找到你电脑上的浏览器（Chrome、Edge、360 浏览器都可以）
2. 点击打开

---

## 步骤 2：访问 Python 官网

1. 在浏览器地址栏（最上面的长条框）点击一下
2. 删除里面的内容
3. 输入以下网址：
   ```
   https://www.python.org/downloads/
   ```
4. 按键盘上的 **Enter（回车）键**

---

## 步骤 3：下载 Python

1. 页面加载后，你会看到一个**黄色按钮**，写着 "Download Python 3.x.x"
2. **点击这个黄色按钮**
3. 浏览器会开始下载（底部或右上角会显示下载进度）
4. 等待下载完成

> ⚠️ **注意**：如果页面是英文的没关系，只要找到黄色下载按钮就行！

---

## 步骤 4：运行安装程序

1. 下载完成后，找到下载的文件
   - 通常在 "下载" 文件夹
   - 文件名类似：`python-3.12.x-amd64.exe`
2. **双击**这个文件

---

## 步骤 5：安装设置（⚠️ 最重要的一步）

安装程序打开后，你会看到一个窗口：

### 5.1 勾选 PATH 选项（必须做！）

在窗口底部，有一个小框：
```
☐ Add python.exe to PATH
```

**用鼠标点击这个小框，让它变成：**
```
☑ Add python.exe to PATH
```

> ⚠️ **这一步非常重要！** 不勾选的话后面会用不了！

### 5.2 开始安装

勾选好后，点击窗口中间的：
```
Install Now
```

---

## 步骤 6：等待安装完成

1. 安装进度条会开始走动
2. 大约需要 2-5 分钟
3. 等待直到看到：
   ```
   Setup was successful
   ```
4. 点击 "Close"

---

## 步骤 7：验证安装（检查是否成功）

### 7.1 打开命令提示符

1. 按键盘上的 **Win + R** 键（同时按）
2. 会弹出一个小窗口
3. 在里面输入：
   ```
   cmd
   ```
4. 按 **Enter（回车）键**
5. 会出现一个黑色窗口

### 7.2 输入检查命令

在黑色窗口中，输入：
```
python --version
```
然后按 **Enter（回车）键**

### 7.3 查看结果

**如果成功**，会显示：
```
Python 3.12.x
```
（x 是数字，只要是 3.10 以上就可以）

**如果失败**，会显示：
```
'python' 不是内部或外部命令...
```
这说明安装有问题，请告诉我！

---

## ✅ 安装成功检查清单

完成后，对照检查：

- [ ] Python 官网下载的（不是 360 软件管家等第三方）
- [ ] 安装时勾选了 "Add python.exe to PATH"
- [ ] 安装完成提示 "Setup was successful"
- [ ] 在 cmd 中输入 `python --version` 显示版本号

**全部打勾** → 恭喜！Python 安装成功！🎉

## 🐛 常见问题

### 问题 1：下载很慢怎么办？

**解决**：Python 官网有时确实慢，可以：
1. 等待一会儿
2. 或者用这个国内镜像：https://www.python.org.cn/downloads/

### 问题 2：安装时提示错误

**可能原因**：
- 电脑没有管理员权限
- 之前安装过 Python 有冲突

**解决**：告诉我具体错误信息

### 问题 3：验证时显示'python'不是命令

**原因**：安装时没有勾选 "Add python.exe to PATH"

**解决**：
1. 卸载 Python（控制面板 → 程序 → 卸载程序）
2. 重新安装
3. **一定要勾选 PATH 选项**

### 问题 4：显示的是 Python 3.9 或更低版本

**原因**：下载的不是最新版本

**解决**：
1. 卸载当前版本
2. 重新去官网下载最新版
3. 安装
